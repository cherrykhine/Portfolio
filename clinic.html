ABOUT THIS PROJECT

A desktop-based Clinic Management System developed using Java, GUI and MySQL to automate clinic operations including users role mangement systems (Admin, Receptionist, Doctor, Pharmacist, Patient), appointment scheduling, billing, and pharmacy inventory. The system implements RBAC - Role-Based Access Control and secure database integration using JDBC.

TECH STACK USED

Java (Swing and AWT) JDBC Java Libraries & APIS NetBeans IDE Java SE MySQL Server SQL
